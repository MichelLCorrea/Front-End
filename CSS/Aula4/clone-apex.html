Conheça o método exclusivo que criamos para ajudar você a compreender de uma vez por todas como funciona a programação de softwares, e transforme-se em um dos profissionais mais desejados pelas empresas de TI.

Quero saber mais sobre esse método

Consultoria gratuita

Consultoria Gratuita para conversar com um especialista e tirar todas as suas dúvidas sobre essa promissora carreira. Uma excelente oportunidade para compartilhar seus sonhos e decidir quais os melhores caminhos e as estratégias certas para realizá-los.

Formação completa de 160 horas

Após a consultoria, se você estiver decidido a se preparar para iniciar na Carreira de Programador, você será acompanhado por um mentor, que estará ao seu lado nas aulas presenciais, te ensinando de maneira totalmente prática e voltada para as necessidades do mercado.

Comunidade exclusiva para Devs

Após finalizar as aulas presenciais, você vai participar de uma Comunidade Exclusiva de Desenvolvedores da Apex. Esta comunidade tem um grupo no Whatsapp, onde os participantes recebem oferta de vagas todas as semanas. Os integrantesda comunidade também participam de palestras e workshops mensais gratuitos para capacitação e networking.

Nesta Formação de 160 horas você aprende:

Lógica de Programação e Banco de dados

Dominar a Lógica de Programação é essencial para que você compreenda como funciona a criação de códigos em qualquer linguagem. Nosso treinamento te ensina a lógica utilizando Javascript, uma das linguagens mais modernas e de fácil compreensão. Estes 2 módulos somam 36 horas de treinamento.

Linguagem de Programação Fundamentos e Avançada

Após compreender a lógica e entender como salvar seus códigos no Banco de Dados, você vai iniciar uma jornada profunda de aprendizagem em uma linguagem de programação, que pode ser Java, C# ou Python, de acordo com a sua preferência. Estes 2 módulos somam 60 horas de treinamento.

Linguagem de Programação para a WEB

No módulo web você aprende como criar sistemas que rodam na web com programas na linguagem de sua escolha: Java WEB, Spring, React, Angular ou ASP.NET Core. Este módulo tem 40 horas de treinamento.

Formação Comportamental

Aprender a programar é essencial para se tornar um programador, porém, para você se destacar no mercado é preciso mais. E é por isso que nós vamos te ensinar conceitos básicos de Gerenciamento Ágil de Projetos, Empreendedorismo e Inovação. E para que você possa crescer na sua carreira vamos te ensinar a entender mais sobre Cultura Organizacional, Inteligência Emocional e Planejamento de Carreira. Estes 6 módulos juntos somam 24 horas de treinamento.
